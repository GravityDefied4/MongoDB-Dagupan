# CSS Basics Lab

## Overview
This project demonstrates fundamental CSS concepts including the box model, flexbox layout, form styling, and basic page structure.

## Features
- HTML and CSS integration
- CSS Box Model (margin, padding, border, width, height)
- Flexbox layout system
- Styled HTML form
- Structured page layout (header, main, footer)

## Concepts Covered
- CSS Box Model
- Flexbox (justify-content, align-items, flex-direction)
- Form styling
- Basic page layout

## Purpose
This project is part of a CSS fundamentals exercise to build understanding of layout and styling techniques.
