# JavaScript Destructuring Lab

## Overview
This project demonstrates ES6 destructuring techniques in JavaScript, including arrays, objects, nested structures, default values, and function parameters.

## Features
- Array destructuring
- Object destructuring
- Nested destructuring
- Default values
- Variable renaming
- Function parameter destructuring

## Concepts Covered
- ES6 syntax improvements
- Cleaner variable extraction
- Simplified function parameters
- Handling complex data structures

## How to Run
1. Open index.html in browser
2. Open Developer Console
3. View outputs in console
