# Debug Faulty Code Lab

## Overview
This project demonstrates how to identify and fix common JavaScript errors including syntax errors, runtime errors, and logical errors.

## Features
- Syntax error debugging
- Runtime error debugging
- Logical error correction
- Console debugging using console.log()
- Functional testing of corrected code

## Error Types Covered

### Syntax Errors
Incorrect JavaScript syntax that prevents code execution.

### Runtime Errors
Errors that occur while the program is running.

### Logical Errors
Mistakes in program logic that produce incorrect results.

## Debugging Techniques Used
- Browser console inspection
- console.log() tracing
- Step-by-step code testing

## How to Run
1. Open index.html in browser
2. Open browser developer console
3. Observe outputs and corrected code behavior
