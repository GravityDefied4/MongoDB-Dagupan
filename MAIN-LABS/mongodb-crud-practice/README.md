# MongoDB CRUD Practice

## Overview
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using MongoDB shell (mongosh).

## Database Used
crudPracticeDB

## Collection
items

## Features Practiced
- Insert single and multiple documents
- Query documents with conditions
- Update single and multiple documents
- Delete documents
- Basic database operations in MongoDB shell

## Tools Used
- MongoDB
- mongosh

## Purpose
This project is part of practice for understanding MongoDB CRUD operations and database manipulation using the shell.
