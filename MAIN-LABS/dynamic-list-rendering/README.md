# Dynamic List Rendering React App

## Overview
This project demonstrates dynamic list rendering in React using the useState Hook and array mapping.

## Features
- Dynamic rendering using map()
- Add new items
- Remove items
- Input validation
- Empty list handling
- React Hooks (useState)

## Concepts Covered
- Functional components
- State management
- Event handling
- Conditional rendering
- Dynamic UI updates

## How to Run

```bash
npm install
npm start
```

## Technologies Used
- React
- JavaScript
- CSS
