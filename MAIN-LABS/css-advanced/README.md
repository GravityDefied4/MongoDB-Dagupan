# CSS Advanced Responsive Layout

## Overview
This project demonstrates a responsive web layout using HTML and CSS. It focuses on mobile-first design, media queries, and CSS Grid.

## Features
- Semantic HTML structure (header, nav, main, section, footer)
- Mobile-first responsive design
- CSS Grid layout system
- Media queries for tablet and desktop views
- Fully responsive navigation and grid system

## Breakpoints Used
- Mobile: default styles
- Tablet: 768px
- Desktop: 1024px

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Media Queries)

## Purpose
This project is part of a CSS advanced layout exercise to understand responsive web design principles.
