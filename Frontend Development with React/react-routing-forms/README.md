# React Routing and Forms Lab

This lab demonstrates React Router navigation and form handling in React.

Features:
- Client-side routing using React Router
- Controlled form components
- Form validation
- Dynamic navigation between pages
- State management using useState

Technologies Used:
- React
- React Router DOM
- JavaScript
- CSS

How to Run:
1. Open terminal
2. Run npm install
3. Run npm start
4. Open http://localhost:3000
