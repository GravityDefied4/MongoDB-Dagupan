API Integration Lab

This lab demonstrates how to fetch data from an external API using React.

Features:
- Fetching API data using fetch()
- Using useEffect and useState hooks
- Displaying loading and error states
- Rendering dynamic user data
- Basic component styling

Challenges Encountered:
One challenge I encountered was understanding asynchronous data fetching and how useEffect works. I solved this by carefully reviewing the flow of fetch(), async/await, and state updates.

Another challenge was handling loading and error states properly. After testing the application multiple times, I understood how conditional rendering works in React.

How to Run:
1. Open terminal
2. Run npm install
3. Run npm start
4. Open http://localhost:3000
