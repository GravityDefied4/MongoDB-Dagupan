# State and Props Lab

This lab demonstrates the use of state and props in React.

Features:
- Counter component using useState
- Greeting component using props
- Passing props from parent to child
- Lifting state up between components
- Dynamic UI updates

Technologies Used:
- React
- JavaScript
- JSX

How to Run:
1. Open terminal
2. Run npm install
3. Run npm start
4. Open http://localhost:3000
